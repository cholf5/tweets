# 2025 年终总结

回头看 2025 年，我做了几次重要的取舍，也亲手验证了一些长期悬而未决的问题。

简单总结今年于我而言最有价值的事：
1. 彻底拥抱 .NET
2. 换了份工作，工资大涨
3. 证明兼职合伙做项目不可行
4. 开始养猫了

这里面有一定有因果关系，正是因为有了1，也才有了2。至于养猫，是纯生活上的事，跟工作无关，但给我提供了很多欢乐。

## 彻底拥抱 .NET

过去我一直给 .NET 和 C# 发“好人卡”，即：“我知道你很好，但就是不用你。”

我也说不上为什么，可能就是工作中没有使用契机。

虽然我过去做了好多年的主程，但上面还有技术总监，在选框架这件事上并没有太多自主权。今年终于有契机让我自主决策了。

#### skynet 和 Lua 的问题
首先就是抛弃了用了很多年的 skynet，并不是它不够好，而是我认为 lua 这种动态语言在项目规模上去后，无法保证可维护性。我曾经试过类似 TypeScript 的方案 Teal，也试过直接在 skynet 里写 ts，最终都以不成熟或没人维护而放弃。

目前唯一成熟且可行的方案就是用 EmmyLua 加海量注解，包括策划配置表和 sproto 协议，都会自动生成注解。代码里的注解就由程序员手动添加了。这虽然能极大提高开发体验，但仍然无法避免变量名写错这类低级错误蔓延到线上的问题。

luacheck 能部分解决拼写错误的问题，但以 lua 的灵活性来说，随便写一些隐晦的代码，它就无能为力了。

用 lua 做游戏后端，始终有一种踩钢丝的感觉，因为不知道哪天就会因为一个变量名写错或者一个 attempt index a nil value 而导致 P0 事故。也因此今年我彻底放弃了 skynet + lua 的组合，开始寻求静态语言的技术栈。

#### 终于不再给 .NET 发“好人卡”了
在充分调研了 Go、Java 和 C# 后，最终选定了 C#。Go 语法过于简陋，Java 则太啰嗦，唯有 C# 语法优雅、生态足够成熟，且和 Unity 可以共享代码。尤其是 Rider 带来的极致的开发体验，这在过去用 lua 时是难以想象的。

上半年我用 .NET 生态里的 Orleans 搭建了一个后端框架，后来被证明是一个彻底失败的东西。首先 Orleans 就过于学院化，在面临真正的工程问题时，往往有很多不如意：
- 比如过度包装，把 Actor 之间的通信封装成函数调用，导致无法在 mailbox 入口做统一的消息拦截。
- 再比如硬生生分出 Silo 和 Client 的概念，导致 Silo 很难主动和 Client 通信。这在 skynet 里是不存在的，因为 skynet 里一切皆 Actor。

虽然 Orleans 不是个令人满意的框架，但经过半年的使用，也让我对 C# 有了一定的开发经验。我对 .NET 生态整体还是非常满意的，且越用越喜欢，甚至爱屋及乌，连带着其生态下的一切都喜欢上了。比如 Blazor 我也很喜欢，现在的头像就是 Blazor 的 Logo。

## 换了份工作

前文说到 Orleans 的彻底失败，其实那个项目在做了半年后也被公司砍了。当然不是因为后端，而是产品方向上让高层不满意。不论如何，我经历了一波裁员，也拿了离职大礼包，算是和平分手。

之后经历了投简历、复习八股、密集面试，顺利找到了新工作，工资也涨了很多（具体数值就不透露了，可能违反保密协议）。

更重要的是，新公司是用 .NET 的。我认为正是因为我对 .NET 的喜爱，以及过去半年的使用，才让我拿到了这个 Offer。当然事后我也知道，面试我的是公司 AI 组的组长，在面试过程中问了我不少 Vibe Coding 的事，我当然是对答如流，因为我就是重度使用者。这显然也加了不少分。

不论如何，能以目前接近40岁的“高龄”，在如此短的时间内，找到一份如此好的工作，令我欣喜不已。本来我已经做好退休打算了，退休城市都选好了。

接下来就是找房子、搬家、解决通勤问题等等，都很顺利，也都让我很满意。

## 证明兼职合伙做项目不可行

今年我尝试跟人兼职合伙做项目，被证明完全行不通。以后我再也不会尝试这种合作形式，也劝大家远离这种模式。

合作形式是：
- 大家不拿工资，签合伙协议，每人占股比例等。
- 利用下班和周末时间兼职开发。
- 远程合作。地理上天南海北，无法线下聚集开发。

这种模式最大的问题是，一份全职工作已经很累了，业余时间本就不多，还想放松放松，结果这样一份没有任何报酬且高风险的兼职，把业余时间也挤占掉了。

结果就是：要么集体摸鱼没有进度、要么人全部疯掉。

> 兼职合伙的问题不在于人不努力，而在于**它同时消耗时间、精力和希望，却不给任何即时反馈**。

我想未来如果我还想做点什么，一定是自己做了。目前 AI 时代不是没有合伙人，AI Agents 就是你的合伙人。低效、笨拙、爱偷懒的人类合伙人是无法和 AI 相提并论的。一个人想干就干，不想干就休息，也比较自由。

> 在 AI 时代，执行力不再稀缺，稀缺的是判断力和持续意愿。
## 开始养猫了

今年工作上虽然进步速度不慢，但在养猫这件事上我可以说是光速进步。从5月份开始喂流浪猫，到年底自己养猫，只经历了半年时间。现在已经是一名光荣的铲屎官啦。

事实证明，养猫真的不难。只需要丁点付出就能换来无尽的快乐。小体型猫吃的也很少，一个月花不了多少钱。

唯一的负面效果可能是：我在家很难专心工作了，时不时就想撸一下它。：）

## 总结

2025年是收获颇丰的一年，人生进度条到这里快过半了，不管是工作、学习、家庭、事业，都处于一个令我满意的状态。经历了这么多事后，我也变得愈发冷静和成熟。马上进入不惑之年了，也确实能感受到能不被外物迷惑，达到心智成熟状态了。

比较遗憾的是，虽然目前 Claude Code 和 Codex 之类的 AI 编程工具越来越强，但我仍然没做出什么像样的游戏或产品。考虑到我同时有一份全职工作和一份兼职，我也和自己和解了。

找到新工作后，兼职已经和平分手，希望2026年我能用业余时间做出一些好玩的独立游戏。
